# Sales-Report-Dashboard
📊 Finance Report Dashboard (Power BI)  This project presents an interactive Finance Report Dashboard built using Power BI, designed to provide clear insights into sales performance, profitability, and business segmentation.
🚀 Key Features
📈 KPIs Overview
Total Gross Sales: 128M
Total Sales: 119M
Total Profit: 17M
Units Sold: 1M
📦 Sales Analysis
Sales breakdown by Product
Sales distribution by Segment (Government, Enterprise, Midmarket, Small Business)
📅 Time-Based Insights
Profit trends over time
Interactive Date Filter (Slicer)
🌍 Geographical Insights
Profit distribution across countries using map visualization
📊 Performance Metrics
Unit Sold Percentage
Sales comparison by Product & Segment
🛠️ Tools & Technologies
Power BI Desktop
Data Modeling
DAX (Data Analysis Expressions)
Interactive Visualizations
🎯 Purpose

This dashboard helps stakeholders:

Monitor financial performance
Identify high-performing products and segments
Analyze profit trends over time
Make data-driven business decisions
